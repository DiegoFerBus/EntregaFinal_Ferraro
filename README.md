# Proyecto E-Commerce (Tienda Drinks)

## Descripción

Este proyecto es una aplicación de e-commerce que permite a los usuarios explorar y comprar bebidas online. Pueden ver detalles de los productos, sus precios y agregar artículos a su carrito. También ajustar las cantidades de los mismos y proceder a la compra.

## Características

- **Catálogo de Productos**: Muestra una lista de productos disponibles con imágenes, descripciones y precios.
- **Detalles del Producto**: Permite a los usuarios ver información detallada sobre cada producto.
- **Carrito de Compras**: Los usuarios pueden agregar productos a su carrito, ajustar las cantidades y eliminar artículos.
- **Total Dinámico**: Se calcula automáticamente el total del carrito basado en los productos y sus cantidades.

## Tecnologías Utilizadas

- React.js
- Bootstrap
- Firebase, Firestore para almacenar y recuperar productos.



### Instala y corre la aplicación

1. **Clonar el Repositorio**

   ```
   git clone <URL_DEL_REPOSITORIO>
   cd <nombreDelRepositorio>
   ```
2. **Instalar Dependencias**
```
npm install
```
3. **Configurar Firebase**

Asegúrate de tener una cuenta de Firebase y haber creado un proyecto.
Configura tu archivo firebase.js con las credenciales de tu proyecto.

4. **Ejecutar la Aplicación**
```
npm start
```
